# Contributing

Thank you for interest in DungeonCrawler Content!

## Getting Started

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Add tests
5. Commit with clear messages
6. Open a Pull Request

## Content Standards

- Use proper Pathfinder 2E terminology
- Follow module documentation
- Test encounters in a DungeonCrawler site
- Include relevant references and source material
